## 二、五大核心板块详解

### 1. 虚拟实验室 (VirtualLab)

**功能定位**：
- 提供电磁驱动霍普金森杆实验的虚拟仿真环境
- 支持2D/3D可视化展示实验全过程
- 允许用户调整实验参数并观察结果

**核心功能**：
- **材料库管理**：7大类材料数据库（金属、聚合物、复合材料、陶瓷、生物材料、地质材料、多孔材料）
- **参数配置**：电压(220-350V)、电流(0-50kA)、脉宽(200-1100μs)、波形类型
- **围压控制**：XYZ三轴独立围压调节(0-200MPa)
- **波形预览**：入射波、反射波、透射波三条曲线实时显示
- **实验动画**：电磁发射→撞击杆加速→应力波传播→试样变形→数据采集

**数据流向**：
```
用户选择材料 → 设置实验参数 → 启动仿真 → 生成波形数据 → 2D/3D动画展示 → 数据采集显示
```

---

### 2. AI智能控制 (AIControl)

**功能定位**：
- 实现三级AI闭环智能优化系统
- 自动优化实验参数以达到最佳测试效果
- 提供算法可视化和训练监控

**核心功能**：
- **LSTM时序预测**：预测应力波传播特性
  - 输入：历史应力波形、材料参数、加载条件
  - 输出：应力波预测、传播时间、衰减系数
  
- **WGAN-GP波形生成**：生成最优驱动波形
  - 输入：目标波形、材料约束、能量限制
  - 输出：优化波形、电容电压、放电时序
  
- **PPO强化学习**：在线优化控制策略
  - 输入：当前状态、目标性能、约束条件
  - 输出：控制动作、策略更新、价值评估

**数据流向**：
```
实验数据 → LSTM预测 → WGAN生成波形 → PPO优化 → 推送至虚拟实验室执行
     ↑                                                    ↓
     └────────────── 反馈实验结果 ────────────────────────┘
```

---

### 3. 多场耦合实验 (MultiField)

**功能定位**：
- 实现热-力-电多场耦合动态加载
- 模拟极端环境条件下的材料响应
- 支持多场同步控制和数据采集

**核心功能**：
- **热场控制**：温度加载(室温-1000°C)
- **力场控制**：应力加载(0-1000MPa)
- **电场控制**：电磁驱动(0-50kA)
- **多场同步**：纳秒级同步控制
- **数据融合**：多物理场数据实时融合

**数据流向**：
```
热场参数 ──┐
力场参数 ──┼→ 多场同步控制器 → 耦合加载 → 多传感器采集 → 数据融合分析
电场参数 ──┘
```

---

### 4. 材料力学分析 (MaterialAnalysis)

**功能定位**：
- 提供材料动态力学性能分析工具
- 支持应力-应变曲线处理和本构模型拟合
- 实现跨材料性能对比和预测

**核心功能**：
- **应力-应变分析**：动态应力-应变曲线处理
- **本构模型拟合**：
  - Johnson-Cook模型
  - Cowper-Symonds模型
  - Zerilli-Armstrong模型
- **性能参数提取**：
  - 动态屈服强度
  - 应变率敏感性
  - 能量吸收特性
- **跨材料对比**：不同材料性能对比分析
- **AI性能预测**：基于材料成分预测动态性能

**数据流向**：
```
实验数据 ─→ 应力-应变曲线 ─→ 本构模型拟合 ─→ 性能参数提取 ─→ 分析报告
                ↓
         AI性能预测模型
```

---

### 5. 系统监控 (SystemMonitor)

**功能定位**：
- 实时监控系统运行状态
- 提供预警和故障诊断功能
- 支持历史数据查询和分析

**核心功能**：
- **实时监控**：
  - 电压、电流、温度
  - 电容储能
  - 电磁干扰(EMI)
  - 系统状态
- **预警系统**：
  - 电压超限预警(>350V)
  - 电容储能预警(>30kJ)
  - 温度超限预警(>80°C)
  - EMI干扰预警(>80dB)
- **故障诊断**：自动诊断系统故障
- **历史数据**：实验历史记录查询

**数据流向**：
```
传感器数据 ─→ 数据采集 ─→ 实时监控 ─→ 阈值判断 ─→ 预警/正常
                              ↓
                         历史数据库
```

## 三、板块间相互影响关系

### 数据流向图

```
                    ┌─────────────────────────────────────┐
                    │           用户操作界面               │
                    └───────────────┬─────────────────────┘
                                    │
        ┌───────────────────────────┼───────────────────────────┐
        │                           │                           │
        ▼                           ▼                           ▼
┌───────────────┐       ┌───────────────────┐       ┌───────────────────┐
│   虚拟实验室   │◄─────►│    AI智能控制      │◄─────►│   多场耦合实验    │
│  (VirtualLab) │       │   (AIControl)     │       │  (MultiField)     │
└───────┬───────┘       └─────────┬─────────┘       └─────────┬─────────┘
        │                         │                           │
        │    ┌────────────────────┘                           │
        │    │                                                │
        ▼    ▼                                                ▼
┌───────────────────┐                               ┌───────────────────┐
│   材料力学分析     │◄──────────────────────────────►│    系统监控       │
│ (MaterialAnalysis)│                               │  (SystemMonitor)  │
└───────────────────┘                               └───────────────────┘
```

### 相互影响说明

#### 1. 虚拟实验室 ↔ AI智能控制
- **正向影响**：虚拟实验室提供实验数据作为AI训练样本
- **反向影响**：AI智能控制优化后的参数推送至虚拟实验室执行
- **协作模式**：形成"实验→学习→优化→再实验"的闭环

#### 2. 虚拟实验室 ↔ 多场耦合实验
- **正向影响**：虚拟实验室的基础参数作为多场耦合的初始条件
- **反向影响**：多场耦合的复杂工况数据反馈至虚拟实验室验证
- **协作模式**：简单实验在虚拟实验室完成，复杂实验需要多场耦合

#### 3. 虚拟实验室 ↔ 材料力学分析
- **正向影响**：虚拟实验室的仿真数据用于材料性能预分析
- **反向影响**：材料力学分析的本构模型参数指导虚拟实验设置
- **协作模式**：材料参数双向流通，相互校准

#### 4. AI智能控制 ↔ 材料力学分析
- **正向影响**：材料力学分析的本构模型作为AI的物理约束
- **反向影响**：AI预测的材料性能用于验证分析模型
- **协作模式**：物理模型+数据驱动双驱动

#### 5. 多场耦合实验 ↔ 系统监控
- **正向影响**：多场耦合实验的传感器数据实时进入系统监控
- **反向影响**：系统监控的预警信息控制多场耦合实验的安全运行
- **协作模式**：监控保障实验安全，实验提供监控数据

#### 6. 系统监控 ↔ 所有板块
- **监控作用**：实时监控所有板块运行状态
- **预警作用**：异常情况及时预警，保护系统安全
- **数据作用**：记录所有操作历史，支持追溯分析

## 四、核心技术栈

### 前端技术
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式**：Tailwind CSS
- **UI组件**：shadcn/ui
- **动画**：Framer Motion
- **3D渲染**：Three.js + React Three Fiber
- **图表**：ECharts
- **状态管理**：Zustand

### AI技术
- **时序预测**：LSTM (Long Short-Term Memory)
- **生成模型**：WGAN-GP (Wasserstein GAN with Gradient Penalty)
- **强化学习**：PPO (Proximal Policy Optimization)
- **语音交互**：百度UNIT + Web Speech API

### 部署技术
- **代码托管**：GitHub + Git LFS
- **前端部署**：Vercel
- **大文件处理**：Git LFS / 云存储CDN

## 五、使用流程示例

### 典型实验流程

```
1. 用户进入虚拟实验室
   └─► 选择材料（如Q235钢）
   └─► 设置实验参数（电压280V、脉宽500μs）
   └─► 启用围压控制（X=50MPa, Y=30MPa, Z=20MPa）
   
2. 启动AI智能优化
   └─► LSTM预测应力波传播
   └─► WGAN生成优化波形
   └─► PPO调整控制策略
   └─► 推送优化参数至虚拟实验室
   
3. 执行虚拟实验
   └─► 2D/3D动画展示实验过程
   └─► 实时显示三条波形曲线
   └─► 采集应力-应变数据
   
4. 材料力学分析
   └─► 处理应力-应变曲线
   └─► Johnson-Cook模型拟合
   └─► 提取动态力学参数
   
5. 系统监控
   └─► 实时监控实验状态
   └─► 预警异常参数
   └─► 记录实验历史
```
