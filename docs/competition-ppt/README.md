## 使用方法

1. **直接复制每个 `##` 块的内容到 PPT** — 表格自动转 PPT 表格
2. **代码块** 用等宽字体复制(推荐 PPT 字体:JetBrains Mono / Consolas)
3. **关键数据**(蓝色加粗的数字)一定要说出来 — 这是评委记忆点
4. **现场可演示** 部分预先用 IDE 打开,答辩时直接切窗口

## 配色建议(与项目主题一致)

| 用途 | 色值 |
|:--|:--|
| 主色(标题) | `#0EA5E9` 科技蓝 |
| 辅助色(数据) | `#22D3EE` 青色 |
| 强调色(警告) | `#F59E0B` 琥珀 |
| 成功色(达成) | `#10B981` 翠绿 |
| 背景 | `#0F172A` 深空蓝 |
| 文字 | `#F1F5F9` 雪白 |

## 字体建议

- 中文标题:**思源黑体 Bold** / 微软雅黑 Bold
- 中文正文:思源黑体 Regular / 微软雅黑
- 英文 + 数字:**Inter** / Roboto / Helvetica
- 代码:JetBrains Mono / Fira Code / Consolas
