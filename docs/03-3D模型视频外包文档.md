# 3D模型视频制作外包需求文档

> 版本: v2.0 | 更新日期: 2026-03-18
> 项目: 数智化电磁驱动霍普金森杆多场耦合动态测试系统

---

## 1. 项目背景

本项目为一个科研级Web实验平台，需要在"虚拟实验室"页面中展示电磁驱动霍普金森杆（SHPB）实验装置的3D效果。

**技术方案**: 采用**预录制3D视频**方案（非实时3D渲染），视频将在网页中通过 `<video>` 标签播放，实现类3D展示效果。

**核心要求**: 视频背景色必须与网站背景色一致（**#0A2540 深蓝色**），确保视频在网页中无缝融合，看起来像"嵌入式3D模型"。

---

## 2. 源模型信息

### 2.1 模型文件
- **文件名**: `new_hopkinson.STEP`
- **格式**: STEP (Standard for the Exchange of Product Data)
- **来源**: SolidWorks/CATIA 工程模型
- **内容**: 完整电磁驱动霍普金森杆实验装置

### 2.2 推荐渲染软件
- **首选**: Blender 4.x（免费、社区活跃、PBR渲染质量高）
- **备选**: SolidWorks Visualize / KeyShot / Cinema 4D
- STEP文件可通过 FreeCAD 转换为 Blender 支持的格式（.obj/.fbx）

## 3. 视频内容需求

共需制作 **4 类视频**，总计 **6-8 段**。

### 3.1 整体旋转展示视频

| 属性 | 规格 |
|------|------|
| 时长 | 30秒，无缝循环 |
| 内容 | 整套装置360°水平旋转展示 |
| 镜头 | 固定中距离，装置居中，轻微俯视（15-20°） |
| 旋转 | 匀速旋转1圈（360°），顺时针 |
| 光照 | 三点布光（主光偏冷色6500K，辅光暖色4000K，轮廓光） |
| 标注 | 关键部件名称标注（中英双语），随旋转出入 |
| 循环 | 首尾帧一致，确保无缝循环播放 |

### 3.2 关键部件特写视频（4段）

每段视频规格：

| 属性 | 规格 |
|------|------|
| 时长 | 每段10-15秒 |
| 镜头 | 从整体平滑推进到部件特写 |
| 标注 | 部件名称 + 关键技术参数标注 |

4段特写内容：

| # | 部件 | 展示重点 | 标注内容 |
|---|------|----------|----------|
| 1 | **三级电磁线圈** | 线圈绕组细节、弹丸通道、三级结构 | "三级电磁线圈 / 3-Stage EM Coil"、级间间距、线圈匝数 |
| 2 | **入射杆与试样夹具** | 杆件精密表面、应变片安装位、试样安装方式 | "入射杆 / Incident Bar"、长径比标注、应变片位置 |
| 3 | **电容器组与RLC电路** | 电容阵列排布、母线连接、高压开关 | "电容器组 / Capacitor Bank"、储能参数、电压等级 |
| 4 | **数据采集系统** | DAQ设备、信号电缆、应变片连接 | "数据采集 / DAQ System"、采样率标注 |

### 3.3 实验过程动画视频

| 属性 | 规格 |
|------|------|
| 时长 | 20秒 |
| 内容 | 模拟完整实验过程的动画 |

**动画时间线**:
```
0-3s:   电容器充电（发光效果渐亮）
3-6s:   电磁线圈依次激活（三级线圈逐级发蓝光）
6-9s:   撞击杆（弹丸）加速发射（运动模糊 + 速度标注）
9-12s:  撞击杆命中入射杆（冲击波纹效果）
12-16s: 应力波沿杆件传播（透明波纹/色彩渐变效果）
16-18s: 试样受压变形（微观放大展示）
18-20s: 数据采集信号（电缆亮起，DAQ屏幕闪烁）
```

**动画技术要求**:
- 发光/辉光效果（Bloom/Glow）表现电磁能量
- 运动模糊表现高速运动
- 透明材质波纹效果表现应力波传播
- 物理级形变动画（金属塑性变形）

### 3.4 爆炸分解视图视频

| 属性 | 规格 |
|------|------|
| 时长 | 15秒 |
| 内容 | 各部件逐一从组装位置飞出，展示内部结构 |

**分解顺序**:
```
0-2s:   整体静态展示
2-5s:   外壳/防护罩分离
5-8s:   电容器组、RLC电路模块分离
8-11s:  线圈、杆件、试样分离
11-13s: 数据采集、冷却系统分离
13-15s: 所有部件悬浮展示（各自轻微旋转）
```

**动画技术要求**:
- 每个部件分离时带标注线引出部件名称
- 分离路径平滑（贝塞尔曲线运动）
- 部件间连接线（虚线）暗示装配关系
- 最终状态可作为静态封面图

## 4. 技术规格

### 4.1 视频格式

| 属性 | 主格式 | 备用格式 |
|------|--------|----------|
| 容器 | WebM | MP4 |
| 编码 | VP9 | H.264 |
| 分辨率 | 1920×1080 (1080p) | 同 |
| 帧率 | 30fps | 30fps |
| 比特率 | 8-12 Mbps | 10-15 Mbps |
| 色彩空间 | sRGB | sRGB |

**为什么需要两种格式**:
- WebM(VP9): 文件更小，Chrome/Firefox/Edge原生支持
- MP4(H.264): Safari/iOS兼容性回退

### 4.2 背景色（关键！）

```
背景颜色: #0A2540
RGB: (10, 37, 64)
说明: 深海蓝，与网站整体背景色完全一致
```

**要求**:
- 整个视频背景必须是**纯色 #0A2540**，无渐变、无纹理
- 模型边缘不能有锯齿导致的浅色像素溢出（需开启抗锯齿）
- 建议使用**透明背景渲染后合成**：先渲染带Alpha通道的视频，再合成#0A2540背景
- 最终效果：在网页中播放时，视频与页面背景无缝融合，看起来像3D模型直接嵌入页面

### 4.3 渲染质量

| 项目 | 要求 |
|------|------|
| 抗锯齿 | 8x MSAA 或等效 |
| 阴影 | 软阴影，分辨率2048+ |
| 反射 | PBR金属反射（不锈钢、铜、铝各自反射率） |
| 环境光 | HDRI环境光（工业/实验室风格） |
| 景深 | 特写视频可添加轻微景深效果 |
| 运动模糊 | 仅实验过程动画使用 |

### 4.4 材质建议

| 部件 | 材质 | 颜色/反射 |
|------|------|-----------|
| 杆件(入射/透射) | 不锈钢/马氏体钢 | 银灰色，高反射 |
| 电磁线圈 | 漆包铜线 | 铜色，中等反射 |
| 电容器外壳 | 铝合金 | 银白色 |
| 高压电缆 | 绝缘橡胶 | 黑色/红色，哑光 |
| 试样 | 根据材料类型 | 可配置 |
| 底座/支架 | 铸铁/工程塑料 | 深灰色 |
| 安全标识 | — | 安全黄 #FFD700 |

## 7. 前端集成方案

### 7.1 视频播放组件

视频将通过 `Model3DVideo.tsx` 组件在网页中播放：

```tsx
// 简化示例
<div className="bg-[#0A2540] rounded-xl overflow-hidden">
  <video
    autoPlay
    loop
    muted
    playsInline
    className="w-full h-full object-contain"
  >
    <source src="/videos/hopkinson-rotate.webm" type="video/webm" />
    <source src="/videos/hopkinson-rotate.mp4" type="video/mp4" />
  </video>
</div>
```

### 7.2 文件存放路径

```
public/
  videos/
    hopkinson-overview.webm       // 整体旋转
    hopkinson-overview.mp4
    hopkinson-coil-detail.webm    // 线圈特写
    hopkinson-coil-detail.mp4
    hopkinson-bar-detail.webm     // 杆件特写
    hopkinson-bar-detail.mp4
    hopkinson-capacitor-detail.webm // 电容特写
    hopkinson-capacitor-detail.mp4
    hopkinson-daq-detail.webm     // DAQ特写
    hopkinson-daq-detail.mp4
    hopkinson-experiment.webm     // 实验过程
    hopkinson-experiment.mp4
    hopkinson-explode.webm        // 爆炸分解
    hopkinson-explode.mp4
    thumbnails/
      hopkinson-overview.png
      hopkinson-coil-detail.png
      ...
```

### 7.3 视频切换交互

网页中提供视频选择菜单：
- "整体展示" — 播放旋转视频

## 8. 时间线与沟通

### 8.1 建议制作流程

```
阶段1 (3天):  STEP文件导入 + 材质设置 + 光照环境搭建
阶段2 (2天):  整体旋转展示视频制作
阶段3 (3天):  4段特写视频制作
阶段4 (4天):  实验过程动画制作（最复杂）
阶段5 (2天):  爆炸分解视图制作
阶段6 (1天):  最终渲染 + 格式转换 + 交付
```

**总计**: 约15个工作日

### 8.2 验收标准

1. ✅ 所有视频背景色精确匹配 #0A2540
2. ✅ 视频边缘无锯齿/溢色
3. ✅ WebM + MP4 双格式可在主流浏览器播放
4. ✅ 循环视频首尾无缝衔接
5. ✅ 标注文字清晰可读
6. ✅ 金属材质反射真实自然
7. ✅ 实验动画流程符合实际物理过程
8. ✅ 文件大小合理（单段视频 < 50MB）
