# SHPB 原理与一维应力波理论

> Split Hopkinson Pressure Bar (SHPB) — 分离式霍普金森压杆

---

## 一、SHPB 实验概述

SHPB 是研究材料在高应变率（10² ~ 10⁴ /s）下动态力学行为的标准实验技术。本平台采用**电磁驱动**方式替代传统气炮驱动，具有精确可控、响应快速的优势。

### 实验装置组成

```
┌──────────┐   ┌──────────┐   ┌──────┐   ┌──────────┐   ┌──────────┐
│ 电容器组  │──►│ 电磁线圈  │──►│打击杆 │──►│  入射杆  │──►│  透射杆  │
│CapacitorBank│ │EM Coil  │   │Striker│   │Incident │   │Transmit │
└──────────┘   └──────────┘   └──────┘   └────┬─────┘   └──────────┘
                                              │
                                         ┌────┴────┐
                                         │  试件   │
                                         │Specimen │
                                         └─────────┘
```

### 实验流程

1. **电容放电**：电容组（4000μF）在设定电压（1000-4000V）下放电
2. **电磁加速**：锥形次级线圈产生轴向电磁力，加速打击杆
3. **撞击入射杆**：打击杆以速度 v 撞击入射杆，产生入射应力波 εᵢ
4. **应力波传播**：入射波到达试件，由于阻抗失配，产生反射波 εᵣ 和透射波 εₜ
5. **数据采集**：应变片采集三波信号，高速相机同步记录

## 二、核心物理公式

### 2.1 电磁驱动

电容储能：

```
E = ½CU²
```

打击杆速度（考虑电磁耦合效率 η ≈ 8%~15%）：

```
v = √(2ηE / m_striker)
```

平台参数：C = 4000μF，η = 12%，打击杆质量由几何参数自动计算。

### 2.2 一维应力波

弹性波速：

```
c = √(E/ρ)
```

波阻抗：

```
Z = ρcA = A√(ρE)
```

入射脉冲峰值应力：

```
σᵢ = ρ_bar · c_bar · v_striker / 2
```

脉冲持续时间（打击杆中往返一次）：

```
T = 2L_striker / c_bar
```

### 2.3 阻抗失配与反射/透射

反射系数：

```
R = (Z_specimen - Z_bar) / (Z_specimen + Z_bar)
```

透射系数：

```
T = 2Z_specimen / (Z_specimen + Z_bar)
```

### 2.4 三波法

从三波信号计算试件力学量：

**试件应力**（由透射波确定）：

```
σₛ(t) = EA/(2Aₛ) · [εᵢ(t) + εᵣ(t) + εₜ(t)]
```

满足应力平衡时简化为：

```
σₛ = (A_bar/A_specimen) · E_bar · εₜ
```

**应变率**（由反射波确定）：

```
ε̇ₛ(t) = c_bar/Lₛ · [εᵢ(t) - εᵣ(t) - εₜ(t)]
```

简化为：

```
ε̇ₛ = -2c_bar · εᵣ / Lₛ
```

**应变**：

```
εₛ(t) = (c_bar/Lₛ) · ∫₀ᵗ [εᵢ - εᵣ - εₜ] dτ
```

### 2.5 应力平衡验证

SHPB 实验有效性判据：

```
R = |σ₁ - σ₂| / max(σ₁, σ₂) < 5%
```

其中 σ₁ = E·A·εᵢ + εᵣ)/Aₛ，σ₂ = E·A·εₜ/Aₛ。

## 四、仿真准确性保障

1. **理论基础**：基于经典一维应力波理论，在 L/D > 20 的条件下高度精确
2. **参数来源**：50+ 材料的 J-C 参数来自已发表的实验文献数据
3. **有效性校验**：内置应力平衡判据 R < 5%
4. **物理约束**：所有计算结果受物理定律约束（能量守恒、波阻抗匹配）
