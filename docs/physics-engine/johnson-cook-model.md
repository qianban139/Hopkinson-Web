# Johnson-Cook 本构模型

> 描述材料在高应变率和高温条件下流动应力的经验本构模型

---

## 一、模型概述

Johnson-Cook (J-C) 模型是动态力学领域最广泛使用的本构模型，通过将应变硬化、应变率效应和热软化三个因素解耦相乘，描述材料的流动应力：

```
σ = (A + Bεⁿ)(1 + C·ln(ε̇/ε̇₀))(1 − T*ᵐ)
     ─────────  ──────────────── ────────
     应变硬化项    应变率效应项     热软化项
```

## 二、参数含义

| 参数 | 单位 | 物理含义 |
|------|------|---------|
| A | MPa | 参考条件下的屈服应力 |
| B | MPa | 应变硬化系数 |
| n | - | 应变硬化指数 |
| C | - | 应变率敏感系数 |
| m | - | 温度软化指数 |
| Tm | °C | 材料熔点 |
| ε̇₀ | /s | 参考应变率（本平台取 1.0 /s） |

### 无量纲温度

```
T* = (T - T_room) / (Tm - T_room)
```

其中 T_room = 25°C。

## 三、三项物理效应

### 3.1 应变硬化项 (A + Bεⁿ)

描述材料随塑性变形增加而强化的行为：
- **A** 决定初始屈服应力
- **B** 和 **n** 控制硬化速率
- n < 1：硬化速率随应变递减（典型金属行为）

### 3.2 应变率效应项 (1 + C·ln(ε̇/ε̇₀))

描述应变率对流动应力的影响：
- C > 0：正应变率敏感性（大多数材料）
- C 越大，材料对加载速率越敏感
- 高分子材料 C 值通常远大于金属

### 3.3 热软化项 (1 − T*ᵐ)

描述温度升高导致材料强度下降：
- T = T_room 时，T* = 0，该项 = 1（无软化）
- T = Tm 时，T* = 1，该项 = 0（完全熔化）
- m 控制软化曲线形状

## 四、平台实现

### 源文件

`src/services/shpbPhysicsEngine.ts` — `johnsonCookStress()` 函数

### 实现细节

```typescript
export function johnsonCookStress(
  strain: number,        // 等效塑性应变
  strainRate: number,    // 应变率 (/s)
  temperature: number,   // 温度 (°C)
  jc: JohnsonCookParams  // J-C 参数
): number {
  // 应变硬化项
  const hardeningTerm = jc.A + jc.B * Math.pow(Math.max(strain, 1e-9), jc.n);
  // 应变率效应项（确保 ε̇/ε̇₀ ≥ 1）
  const rateTerm = 1 + jc.C * Math.log(Math.max(strainRate / 1.0, 1));
  // 热软化项（T* 限制在 [0, 1]）
  const T_star = Math.max(0, Math.min(1, (temperature - 25) / (jc.Tm - 25)));
  const thermalTerm = 1 - Math.pow(T_star, jc.m);

  return hardeningTerm * rateTerm * thermalTerm;
}
```

## 六、动态增强因子 (DIF)

DIF 是评价材料应变率敏感性的重要指标：

```
DIF = σ_dynamic / σ_static
```

| 材料类型 | 典型 DIF 范围 | 应变率敏感性 |
|---------|--------------|------------|
| 金属 | 1.1 ~ 2.0 | 中等 |
| 岩石/混凝土 | 2 ~ 5 | 高 |
| 高分子 | 2 ~ 10 | 很高 |
| 陶瓷 | 1.5 ~ 3.0 | 中等 |

## 七、模型局限性

1. **三因素可分离假设**：J-C 假设应变、应变率和温度效应可独立相乘，在极端条件下可能失效
2. **无损伤演化**：不包含损伤项，无法预测断裂
3. **参数拟合依赖**：参数精度依赖于实验数据质量
