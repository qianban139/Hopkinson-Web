# 用户操作指南

> 平台功能介绍与操作流程

---

## 一、平台定位

**数智化电磁驱动霍普金森杆多场耦合动态测试系统** 是一个软硬件一体化的智能实验平台：

- **真实设备 + 真实数据**：可连接真实霍普金森杆装置采集数据
- **AI 智能辅助**：全流程 AI 引导，从参数设置到结果分析
- **教学 + 科研双用途**：服务初学者和资深研究员

## 二、面向用户

### 科研人员

**目标**：在多种参数和环境条件下获取材料动态力学性质

**典型流程**：
1. 选择目标材料
2. 配置实验环境（温度、围压等）
3. AI 助手优化参数
4. 执行实验（虚拟或真实）
5. 进入材料分析页生成报告

### 学生 / 初学者

**目标**：学习霍普金森杆原理与实验过程

**典型流程**：
1. 进入教学系统学习理论
2. 浏览知识图谱了解概念关系
3. 完成在线测验巩固知识
4. 在 AI 助手「小智」引导下完成虚拟实验
5. 理解每一步的物理意义

## 三、页面导览

### 3.1 首页 (`/`)

**核心信息**：
- 平台概述
- 五大模块入口
- 核心功能展示
- 在线状态指示

### 3.2 虚拟实验室 (`/lab`)

**核心功能**：
- 材料选择（30+ 种材料）
- 参数配置（电压、电流、脉宽、波形）
- 围压设置（XYZ 三轴 0-200 MPa）

- 实时三波波形显示
- 实验结果导出

**操作流程**：
```
选材 → 设参数 → 安全检查 → 启动实验 → 查看结果
```

### 3.3 AI 智能控制 (`/ai`)

**核心功能**：
- LSTM 时序预测

- PPO 强化学习优化
- 神经网络可视化
- 训练动画

**操作流程**：
```
选择优化目标 → 启动优化 → 查看三阶段结果 → 应用最优参数
```

### 3.6 系统监控 (`/monitor`)

**核心功能**：

**核心功能**：

- 详细学习视图

## 四、AI 助手「小智」

平台原生 AI 助手，常驻于浮动球入口。

### 启动方式

| 方式 | 说明 |
|------|------|
| 点击浮动球 | 鼠标点击右下角 AI 球 |
| 语音唤醒 | 说「小智」唤醒（需开启麦克风权限） |
| 键盘快捷键 | （可配置） |

### 三种工作模式

1. **聊天模式 (chat)**：自由问答
2. **实验模式 (experiment)**：引导完成完整实验
3. **教学模式**：解释原理、引导学习

### 支持的指令

- 「打开虚拟实验室」→ 程序化导航
- 「选择 5A06 铝合金」→ 设置材料
- 「电压调到 2500」→ 设置参数
- 「开始实验」→ 启动工作流
- 「解释一下 J-C 模型」→ 知识问答
- 「紧急停止」→ 中止实验

## 六、常见问题

### Q: AI 助手不响应？
A: 检查 `.env` 中的 `VITE_LLM_API_KEY` 是否配置正确。

### Q: 语音功能不工作？
A: 需要 HTTPS 环境 + 浏览器麦克风权限。火山引擎 TTS 需要配置 `VITE_VOLCANO_*` 变量。

### Q: 实验结果与文献不符？
A: J-C 模型是简化经验模型，与真实材料存在系统偏差。本平台用于参数搜索范围缩小，最终结果需以真实物理实验为准。

### Q: 浏览器卡顿？
A: 3D 视图开销较大，可切换至 2D 视图。建议使用 Chrome/Edge 最新版。
