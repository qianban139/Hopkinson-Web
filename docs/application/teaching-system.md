# 教学系统指南

> 知识图谱 + 学习路径 + 在线测验

---

## 一、设计理念

教学系统服务于初学者和中级学习者，将 SHPB 实验领域的复杂知识结构化、可视化、互动化。

## 二、文件结构

```
src/features/teaching-system/
├── knowledgeData.ts        # 知识图谱数据
├── useTeachingStore.ts     # 教学状态管理
├── KnowledgeGraph.tsx      # 知识图谱可视化
├── QuizPanel.tsx           # 在线测验面板
├── StudyView.tsx           # 学习视图（详细内容）
└── LearningPaths.tsx       # 学习路径
```

## 三、知识图谱

### 节点类型

| 类型 | 示例 |
|------|------|
| 概念 (Concept) | 应力波、应变率、阻抗 |
| 公式 (Formula) | Johnson-Cook 方程、三波法 |
| 装置 (Device) | 打击杆、入射杆、应变片 |
| 材料 (Material) | 5A06 铝、Q235 钢 |
| 实验方法 (Method) | SHPB、SHTB、SHBB |

### 边关系

| 关系 | 含义 |
|------|------|
| `is_a` | 类属关系 |
| `uses` | 使用关系 |
| `derives_from` | 推导关系 |
| `related_to` | 相关关系 |
| `prerequisite` | 先修关系 |

### 交互特性

- **拖拽节点**：调整布局
- **缩放图谱**：滚轮缩放
- **点击节点**：查看详情
- **高亮路径**：显示节点间关系链
- **搜索定位**：快速跳转到指定概念

## 四、学习路径

### 入门路径（初学者）

```
什么是动态力学
    ↓
应力波基础
    ↓
霍普金森杆装置
    ↓
基本实验流程
    ↓
读懂应力-应变曲线
```

### 进阶路径（中级）

```
一维应力波理论
    ↓
三波法数据处理
    ↓
Johnson-Cook 本构模型
    ↓
应变率敏感性
    ↓
实验有效性判据
```

### 高级路径（研究方向）

```
本构模型对比
    ↓
多场耦合实验
    ↓
高速图像 DIC 分析
    ↓
能量吸收与破碎机制
    ↓
AI 辅助参数优化
```

## 五、在线测验

### 题型

- **单选题**：基础概念
- **多选题**：复杂知识点
- **填空题**：公式和参数
- **判断题**：常见误区
- **计算题**：公式应用

### 难度分级

| 难度 | 适合人群 | 题目数 |
|------|---------|--------|
| 入门 | 完全新手 | 10 |
| 基础 | 学过一遍 | 20 |
| 进阶 | 熟练掌握 | 30 |
| 挑战 | 深度研究 | 15 |

### 评分与反馈

- 答错时显示正确答案 + 详细解析
- 引导跳转到相关知识图谱节点
- 错题自动加入复习列表

## 六、与 AI 助手联动

教学系统与「小智」深度集成：

| 用户行为 | AI 响应 |
|---------|--------|
| 阅读概念时遇到不懂 | 「这个概念怎么理解？」→ AI 详细解释 |
| 测验答错 | 「为什么是这个答案？」→ AI 分析推理 |
| 完成基础学习 | AI 主动建议「要不要试试虚拟实验？」 |
| 学习路径阻塞 | AI 推荐先修知识点 |

## 七、与虚拟实验室联动

```
教学系统学习理论
    │
    ▼
进入虚拟实验室
    │
    ▼
AI 引导按理论流程操作实验
    │
    ▼
实验结果对照理论预测
    │
    ▼
回到教学系统巩固知识
```

形成「学 → 做 → 反思」的闭环。
