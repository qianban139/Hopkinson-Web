function WaveformChart({ data }) {

## 3. 2D Canvas 渲染指南

### 3.1 架构概览

2D实验可视化使用分层Canvas架构：

```
Layer 0: 背景层（网格、标注）      — 静态，仅渲染1次
Layer 1: 设备层（电容器、杆件等）   — 状态变化时更新
Layer 2: 动画层（应力波、粒子）     — 每帧更新@60fps
Layer 3: HUD层（标注、数字读数）    — 数据变化时更新
```

### 3.2 设备渲染器接口

```typescript
// features/experiment-2d/renderers/types.ts

interface DeviceRenderer {
  /** 设备在Canvas中的位置和尺寸 */
  bounds: { x: number; y: number; width: number; height: number };

  /** 渲染静态设备（设备层） */
  renderStatic(ctx: CanvasRenderingContext2D): void;

  /** 渲染动画帧（动画层） */
  renderAnimation(ctx: CanvasRenderingContext2D, phase: AnimationPhase, progress: number): void;

  /** 渲染标注（HUD层） */
  renderAnnotations(ctx: CanvasRenderingContext2D): void;
}

type AnimationPhase = 'idle' | 'charging' | 'coilAccel' | 'strikerLaunch'
  | 'wavePropagate' | 'deformation' | 'dataCollect';
```

### 3.3 金属渐变渲染

```typescript
// 圆柱体金属渐变（模拟光照反射）
function createMetallicGradient(
  ctx: CanvasRenderingContext2D,
  x: number, y: number, width: number, height: number,
  baseColor: string // 如 '#B0B0B0' 钢灰色
): CanvasGradient {
  const gradient = ctx.createLinearGradient(x, y, x, y + height);
  // 7色阶：暗→亮→暗 模拟圆柱体截面光照
  gradient.addColorStop(0.0, darken(baseColor, 0.3));
  gradient.addColorStop(0.15, darken(baseColor, 0.1));
  gradient.addColorStop(0.3, baseColor);
  gradient.addColorStop(0.45, lighten(baseColor, 0.3));  // 高光
  gradient.addColorStop(0.6, lighten(baseColor, 0.15));
  gradient.addColorStop(0.8, baseColor);
  gradient.addColorStop(1.0, darken(baseColor, 0.25));
  return gradient;
}
```

### 3.4 性能优化

```typescript
// 使用 OffscreenCanvas 缓存静态层
const staticLayer = new OffscreenCanvas(canvasWidth, canvasHeight);
const staticCtx = staticLayer.getContext('2d')!;

// 仅在初始化或窗口resize时重绘静态层
function renderStaticLayer() {
  staticCtx.clearRect(0, 0, canvasWidth, canvasHeight);
  renderers.forEach(r => r.renderStatic(staticCtx));
}

// 动画循环仅重绘动画层
function animationLoop() {
  // 合成静态层
  mainCtx.drawImage(staticLayer, 0, 0);
  // 绘制动画
  renderers.forEach(r => r.renderAnimation(mainCtx, currentPhase, progress));
  requestAnimationFrame(animationLoop);
}
```

### 5.2 API开发模式

```typescript
// server/routes/materials.ts

import { Router } from 'express';
import { z } from 'zod';
import { prisma } from '../app';

const router = Router();

// 查询参数验证
const listQuerySchema = z.object({
  category: z.string().optional(),
  search: z.string().optional(),
  page: z.coerce.number().int().positive().default(1),
  limit: z.coerce.number().int().positive().max(100).default(20),
});

// GET /api/materials
router.get('/', async (req, res) => {
  const query = listQuerySchema.parse(req.query);

  const where = {
    ...(query.category && { category: query.category }),
    ...(query.search && {
      name: { contains: query.search, mode: 'insensitive' as const }
    }),
  };

  const [data, total] = await Promise.all([
    prisma.material.findMany({
      where,
      skip: (query.page - 1) * query.limit,
      take: query.limit,
      orderBy: { name: 'asc' },
    }),
    prisma.material.count({ where }),
  ]);

  res.json({ data, total, page: query.page, limit: query.limit });
});

export default router;
```

## 8. 常见问题

### Q: 开发时修改了材料数据怎么办？
修改 `src/data/materials.json` 后前端自动热更新。后端部署后需要重新运行 seed 脚本。

### Q: AI对话没有响应？
1. 检查 `.env` 中的 API Key 是否正确
2. 检查网络是否能访问 LLM API（可能需要代理）
3. 查看浏览器控制台错误信息

### Q: 2D Canvas动画卡顿？
1. 检查是否正确使用了分层渲染（静态层 + 动画层）
2. 确认 OffscreenCanvas 缓存生效
3. 减少每帧文本渲染（文本渲染开销大）
4. 使用 Chrome DevTools Performance 面板定位瓶颈

### Q: WebSocket连接失败？
1. 检查 Nginx WebSocket 代理配置（Upgrade/Connection headers）
2. 确认 `proxy_read_timeout` 足够长（WebSocket长连接）
3. 检查 CORS 配置

### Q: 如何添加新材料？
1. 前端: 在 `materials.json` 中添加新条目
2. 后端: 通过 `POST /api/materials` 接口或直接在 Prisma Studio 中添加
3. 必须包含: name, category, density, elasticModulus, yieldStrength, johnsonCook 参数

### Q: 如何添加新的AI可执行操作？
参考第4节"AI助手扩展指南"，在 `aiActionRegistry.ts` 中注册新动作，并添加对应的 `data-ai-target` 属性到目标UI元素。
