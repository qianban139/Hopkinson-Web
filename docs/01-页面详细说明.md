# 数智化电磁驱动霍普金森杆系统 — 各页面详细说明

> 版本: v2.0 | 更新日期: 2026-03-18

---

## 1. 首页 (Home) — `/`

### 定位
系统门户页，向访客展示项目定位、核心能力与技术亮点。

### 核心功能
- 项目标题与 Slogan 展示
- 神经网络粒子背景动画，体现"AI感"
- 核心技术模块卡片导航（虚拟实验室、AI控制、多场耦合、材料分析、系统监控）
- 技术参数概览（电压范围、应变率、采样率等）

### 技术规格
- 纯展示页，无状态管理
- Framer Motion 入场动画
- 响应式布局（移动端适配）

## 2. 虚拟实验室 (VirtualLab) — `/lab`

### 定位
系统核心操作页面，模拟电磁驱动霍普金森杆实验全过程。

### 核心功能

#### 2.1 材料库（左侧 320px 面板）
- 7大类材料手风琴式分类浏览（金属、岩石、混凝土、陶瓷、聚合物、泡沫、复合材料）
- 全文搜索（名称、子类别）
- 分类筛选标签
- 选中材料高亮，显示密度/弹性模量/屈服强度

#### 2.2 2D 真实模型（中央区域）
- **Canvas 2D 渲染引擎**（1100×340 设计分辨率，DPI 自适应）
- 10 个设备渲染器：电容器组、电磁线圈、撞击杆、入射杆、试样、透射杆、围压装置、DAQ系统、动量阱
- 7 阶段动画状态机：idle → charging → coilAccel → strikerLaunch → wavePropagate → deformation → dataCollect
- 金属渐变（7色阶）、应力波可视化、工程标注、网格背景

#### 2.3 3D 视图
- 基于 `<video>` 标签播放预录制视频（替代 Three.js 实时渲染）
- 视频背景色 #0A2540 与卡片背景融合
- 支持播放/暂停/进度控制

#### 2.4 参数控制（右侧 320px 面板）
- **4 种参数预设**：标准SHPB、高应变率、岩石冲击、低速测试
- **AI推荐参数**：每个参数旁显示AI推荐值（紫色），一键应用
- 滑块+输入框组合控件（SliderInputCombo）
- 参数范围：电压 220-350V，电流 0-50kA，脉宽 200-1100μs
- 波形类型选择（正弦/方波/三角/脉冲）
- 围压控制（XYZ 三轴，0-200MPa）
- 实时计算参数（储能 kJ、子弹初速 m/s）

#### 2.5 实时波形显示
- ECharts 三通道波形（入射波σᵢ、反射波σᵣ、透射波σₜ）
- 与 2D 动画同步逐点构建
- 实验完成后显示完整波形

### AI 集成点
- `data-ai-target="lab-voltage"` / `lab-current` / `lab-pulseWidth` — AI可调参数
- `data-ai-target="lab-applyAI"` — 应用AI推荐
- 监听 `ai-start-experiment` 事件 — AI启动实验
- 监听 `ai-set-view-mode` 事件 — AI切换2D/3D

### 数据流
- **输入**: 系统监控（安全检查状态）、AI优化（推荐参数）
- **输出**: 实验数据 → DataBus → AI控制 + 材料分析

## 3. AI 智能优化控制 (AIControl) — `/ai`

### 定位
展示 AI 三级优化管线如何分析实验数据并给出优化建议。回答"AI是如何帮你优化的"。

### 核心功能

#### 3.1 优化流水线可视化
- 6 节点管线图：VirtualLab数据 → LSTM → WGAN-GP → PPO → 优化参数 → 返回VirtualLab
- 节点可点击跳转到对应详情区域
- 活跃节点脉冲发光

#### 3.2 AI 闭环优化循环
- SVG 圆形拓扑：数据采集 → LSTM预测 → WGAN生成 → PPO优化 → 参数应用 → 实验验证
- 动画流动光点

#### 3.3 三级算法卡片
- **LSTM 时序预测**：隐藏层、时间步长、学习率、批大小
- **WGAN-GP 波形生成**：生成器层数、判别器迭代、梯度惩罚、噪声维度
- **PPO 策略优化**：裁剪比率、折扣因子、GAE lambda、策略迭代
- 每个卡片：训练曲线图(ECharts)、I/O标签、超参数滑块、训练/停止按钮

#### 3.4 优化前后对比
- 原始参数 vs AI优化参数并列对比
- 4 项改进指标（应力均匀度+23%、能量效率+15%、峰值精度+18%、波形质量+12%）
- "应用优化参数"按钮 → 写回 VirtualLab

#### 3.5 AI 决策洞察卡片
- 3 张卡片解释 WHY：为什么调整电压/电流/脉宽
- 置信度百分比条
- 影响指标标注

#### 3.6 波形对比图
- ECharts: 原始波形(虚线) vs AI优化波形(实线+面积)

#### 3.7 优化历史时间线
- 最近6次优化记录
- 时间、参数、改进百分比

#### 3.8 数据流拓扑
- DataBusVisualization 组件：5 模块节点 + 6 数据边 + 流动动画

### AI 集成点
- `data-ai-target="ai-startOptimization"` — AI触发全流程优化

### 数据流
- **输入**: VirtualLab（实验数据）
- **输出**: 优化参数 → DataBus → VirtualLab

## 4. 多场耦合分析 (MultiField) — `/multifield`

### 定位
极端环境模拟器，回答"这个材料在极端条件下会怎样？"

### 核心功能

#### 4.1 场景预设
- 4 种工程场景卡片：深部矿井、航空航天、核反应堆、电动汽车电池碰撞
- 选择场景自动预设温度/应力/电磁场参数

#### 4.2 三场耦合关系图
- SVG 等边三角形拓扑：温度场(T) ↔ 应力场(σ) ↔ 电磁场(B)
- 6 条双向耦合效应标注（热膨胀、绝热生热、洛伦兹力、磁弹效应等）
- 节点大小/发光随参数强度变化
- 中心显示"耦合度 X%"

#### 4.3 场参数控制
- 温度场：20-1000°C（热软化提示）
- 应力场：0-2000MPa（载荷等级提示）
- 电磁场：0-100T（涡流效应提示）

#### 4.4 场分布可视化
- 3 个 SVG 可视化面板（温度热图、应力矢量、电磁场线）
- 动画等温线、应力椭圆、EM脉冲波

#### 4.5 结果对比面板（新增）
- ECharts: 标准SHPB曲线(蓝) vs 多场耦合曲线(红)
- 量化偏差注释："忽略耦合效应偏差 X%"
- 热软化、涡流能耗、总体偏差三项指标

#### 4.6 耦合效应分析
- 热-力/力-电磁/热-电磁 耦合强度条
- 三场全耦合时显示总强度

### AI 集成点
- `data-ai-target="multifield-scenarios"` / `multifield-params` / `multifield-run`
- 监听 `ai-select-scenario` / `ai-set-multifield-params` / `ai-run-coupling` 事件

### 数据流
- **输入**: VirtualLab（基准数据）
- **输出**: 耦合数据 → DataBus → 材料分析

## 5. 材料力学分析 (MaterialAnalysis) — `/analysis`

### 定位
最终结果页面，实验数据的深度分析与AI预测。

### 核心功能

#### 5.1 材料数据库侧边栏
- 搜索 + 分类筛选（全部/金属/矿石/混凝土/陶瓷/聚合物/泡沫/复合材料）
- 对比模式（最多4种材料同时对比）
- J-C参数快速预览

#### 5.2 实验数据集成横幅（新增）
- 顶部显示上次实验信息（材料名、电压、峰值应力、应变率）
- "查看实验"链接 + "AI分析此材料"快捷按钮

#### 5.3 分析面板（5个Tab）
- **应力-应变曲线**: ECharts多材料叠加，支持高亮/对比
- **J-C拟合可视化**: Johnson-Cook模型参数可视化，R²拟合度
- **性能雷达**: 多维性能雷达图（密度、模量、强度、韧性等）
- **AI预测**: 输入应变率+温度，预测动态力学响应
- **参数总表**: 全材料J-C参数表格，支持CSV/JSON导出

#### 5.4 数据导出
- CSV / JSON / 图片(PNG) 导出
- 图表高分辨率导出（2x像素比）

### AI 集成点
- `data-ai-target="analysis-predict"` — AI触发预测
- `data-ai-target="analysis-export"` — AI触发导出

### 数据流
- **输入**: VirtualLab（波形数据）、多场耦合（耦合数据）
- **输出**: 分析报告

## 6. 系统监控 (SystemMonitor) — `/monitor`

### 定位
安全守护者，实验前必检项，确保设备状态安全。

### 核心功能

#### 6.1 安全检查清单
- 7 项检查：电容组、冷却系统、EMI屏蔽、试件对位、数据采集、紧急系统、人员安全
- 每项模拟传感器读数 + 通过/警告/失败状态
- 全部通过后解锁虚拟实验室

#### 6.2 实时仪表盘
- 5 个圆形仪表：电压(V)、电流(kA)、储能(%)、温度(°C)、EMI(dB)
- 平滑动画过渡（useSmoothValue hook）
- 警告/危险阈值颜色变化

#### 6.3 趋势图
- 3 通道 ECharts 趋势（电压、电流、温度），30秒滚动窗口

#### 6.4 预警中心
- 实时检测超阈值参数
- 按危险等级分类（正常/警告/危险）
- 可配置阈值滑块

#### 6.5 紧急停止
- 红色大按钮，一键断电
- 触发全局告警

### AI 集成点
- `data-ai-target="monitor-safetyCheck"` — AI执行安全检查
- `data-ai-target="monitor-emergencyStop"` — AI紧急停止

### 数据流
- **输出**: 安全检查状态 → DataBus → VirtualLab（解锁实验）
